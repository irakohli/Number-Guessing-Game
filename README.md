# Number Guessing Game 🎯

A simple Python game where the computer picks a random number and you have to guess it.

## How to Play
- The computer selects a random number between 0 and 100
- You keep guessing until you get it right
- The game tells you if your guess is too high or too low
- You can play again after winning

## How to Run
Make sure Python is installed, then run:
python3 noguessgame.py

## Tech Used
- Python 3
- random module
